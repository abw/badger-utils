    public inflect(n: Integer, singular: String, plural: String, no: String): * source 

    import {inflect} from '@abw/badger-utils'

    Utility function to inflect a string passed as the second argument to the singular/plural form based on the number passed as the first argument. Uses the {@link pluralise()} function which only works on words with standard pluralisations. The third argument can be provided as the plural form where necessary. The optional 4th argument can be used to provide a different word for the case where n is 0. The default is "no", e.g. "no badgers" when n is 0.

    Params:

    NameTypeAttributeDescription
    n Integer

    number of items
    singular String

    singular form
    plural String
    • optional

    optional plural form
    no String
    • optional
    • default: 'no'

    optional word to use when n is 0

    Return:

    *

    Example:

    inflect(0, 'cat')     # no cats
    inflect(1, 'cat')     # 1 cat
    inflect(2, 'cat')     # 2 cats
    inflect(0, 'child', 'children')     # no children
    inflect(1, 'child', 'children')     # 1 child
    inflect(2, 'child', 'children')     # 2 children
    inflect(0, 'black', 'black', "none, none more")     # none, none more black
